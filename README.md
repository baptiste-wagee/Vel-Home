# Vel'Home

**L'habitat mobile, durable et lumineux.**

Vel'Home est un site vitrine présentant des Tiny Houses compactes, légères et tractables à vélo. Le projet met en avant un mode de vie durable, mobile et accessible, à travers un catalogue de modèles, une présentation de l'équipe et un formulaire de contact.

## Aperçu

- 🏠 Présentation des Tiny Houses et des valeurs de la marque (Écologie, Solidarité, Mobilité, Inclusion)
- 📖 Catalogue des modèles disponibles
- 👥 Page "À propos" présentant l'équipe et les fondateurs
- ✉️ Page de contact
- 🔐 Page de connexion

## Structure du projet

```
Vel-Home/
├── Projet/
│   ├── html/          # Pages du site (index, catalogue, contact, connexion, à propos)
│   ├── css/            # Feuille de style (style.css)
│   ├── img/             # Images, logo et icônes
│   └── video/           # Vidéos de présentation
├── Annexes/            # Documents complémentaires (mails, captures vidéo)
├── Capture d'écran/     # Maquettes du site
└── Projet.zip           # Archive du projet
```

## Technologies

- HTML5 / CSS3
- Polices : [Montserrat](https://fonts.google.com/specimen/Montserrat) et [DM Serif Display](https://fonts.google.com/specimen/DM+Serif+Display) (Google Fonts)

## Lancer le projet

Le site est statique, il suffit d'ouvrir le fichier `Projet/html/index.html` dans un navigateur, ou de servir le dossier `Projet` avec un serveur local (ex : extension "Live Server" de VS Code).
